Roll your WEBT Task

Now that you have learned the basics for a couple of web technologies, you'll receive a custom exercise to complete. Bonus tasks also involve technologies which you might not have put into practice yet.
For generating your own custom exercise you'll be asked to draw from a pile of playing cards. The card meanings are as following:

Scenario - Draw 1 card

Card Scenario
Ace You are tasked with organizing the event page for a rave - the Unicorn Zombie Apocalypse, As well as presenting the date and time of the rave on a startpage, people have to be able to put their own name on the guest list via a website form, along with their t-shirt size (XS - XXL). To represent the great success of the event the start page has to feature a counter of the number of enlisted guests.
https://www.youtube.com/watch?v=UyI4v5sxT54
2 The representatives of a local golf club - the "Arnold Palmer Fan Boys & Girls" - have tasked you to create a webapp for getting a course overview and ideally tracking their score. Hole number, Hole name, PAR Score per hole. Optional: Ideally players are able to enter their name and track their scores for each hole.
https://en.wikipedia.org/wiki/Arnold_Palmer
3 A local natural park (the "Lainzer Tiergarten") wants to increase perceived safety of its visitors by giving them the possibility of tracking a boar attack. You are to develop the "I got boared" webapp, which should empower its customers to track date, time and their geolocation via a single button press. Optional: Render statistics about all boar attacks broken down to months, days, etc. "Is it dangerous to wander the forest on a wednesday? Do more boars come out at night? I got boared will tell you the truth!"
https://www.lainzer-tiergarten.at/
4 Cookies for Future - a local chinese restaurant wants to help reducing produced waste by switching to digital fortune cookies. You are to develop a webapp in which users receive a random fortune cookie text by clicking on a button. Optional: make the service IP resistant, so greedy fortune hunters are unable to pull an arbitrary number of fortune texts.
https://en.wikipedia.org/wiki/Fortune_cookie
5 Pirate-Crew Constructor - the local pirate harbour Tortuga has asked you to create a pirate crew constructor, where you can create a new ship and assign crewmembers.
https://www.youtube.com/watch?v=y9bXjttMxKY
6 Pushup-Tracker - a local fitness club has asked you to make pushups trackable with your smartphone. You should be able to start an activity which is then tracked with date and time and tracks the number of pushups via motion sensor/javascript. After completion the result is to be stored in a database.
https://en.wikipedia.org/wiki/Push-up
7 Carvinal Costume Store - a local store has asked you to enable them to create and list a number of costumes along with their lending price, which can then in turn be lended by users. (e.g.: As user I can select 2 costumes from a dropdown list and the total lending price is calculated for me)
https://www.youtube.com/watch?v=VSaDPc1Cs5U
8 Art Locator - A consortium of museums has contacted you to create a web app which allows to store information on which portrait is located at which venue. (e.g.: Mona Lisa at the Louvre). It is important to be able to store, retrieve and sort the information.
https://www.louvre.fr/en
9 Tour location site - The heavy metal band Pro-Pain has asked you to create a website for their European tour. The tour manager shall be able to select a day via a datepicker, along with a free text field for entering a venue. Ideally the venue field would feature a lookup mechanism which allows you to select an existing city and displays the tourdates along with a map afterwards.
https://www.youtube.com/watch?v=6iLqD4Jo9wg
10 Meme Generator - You are tasked with creating a meme generator. Users should be able to create a new Meme by entering an image link from the web, text to be put at the top and text to be put below the image. The meme is then to be saved in a database and to be rendered on a dedicated page via passing the ID.
https://metro.co.uk/wp-content/uploads/2019/03/SEC_57712924.jpg?quality=90&strip=all
